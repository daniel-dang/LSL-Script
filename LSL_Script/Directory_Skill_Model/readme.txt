The script in this folder will fetch all information of the student's skill
	from SharePoint to the OpenSim. The config script will set the height and
	position of the building prim to be as tall as the student's 
	skill value. Skill value only go from 1 - 4. 

Note**: All URL in the script has been deleted to keep privacy for the 
	Discovery Lab. Drop the script inside of the object will not work.
